Infi Gauge - A more accurate measuring gauge

About Infi Gauge

Infi Gauge is a precise, tipping bucket technology based I2C device used for measurement & monitoring rain rate and total rainfall. With built-in RTC, Alarm System, Data Logging feature and various configuration modes Infi Gauge is perfect rain sensing system choice


Features

  • I2C based
  • Range: 0-700 mm/hour
  • Accuracy: ± 2% 25-500 mm/hour, ± 1 tip from 1-25 mm/hour
  • Alarm System
  • Various Configuration modes

Applications

  • Rain monitoring.
  • Rain measurement.
